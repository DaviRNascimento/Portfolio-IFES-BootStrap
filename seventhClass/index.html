O que aprendemos:

  • divide o espaço horizontal em 12 espaços iguais
  • justify-content funciona como em um container flex-box, usado na div row
  • order define a ordem de apresentação do element
  • ms-auto coloca a margem esquerda como automatica
  • mx-auto define ambas as margens como automaticas
  • mx-auto pode ser usado para alinhar items horizontalmente sem a necessidade de usar justify-content
  • offset(deslocamento) determina quantas colunas estarao vazias antes da apresentação do elemento
  • a classe w-100 adiciona uma quebra de linha
